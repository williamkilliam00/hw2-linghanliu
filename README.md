# HW2: LLM Workflow - Action Item Extractor

## Project Overview
A Python-based tool that uses Google Gemini API (with a robust fallback mechanism) to extract names and tasks from meeting notes.

## How to Run
1. Ensure you have Python 3.10+ installed.
2. Install dependencies: `pip install requests`
3. Add your Gemini API Key to `app.py`.
4. Run the script: `python app.py`

## Features
- **API Integration**: Connects to Gemini LLM.
- **Robustness**: Includes a demo fallback mode if API connection fails.
- **Formatting**: Outputs clean, structured task lists.
